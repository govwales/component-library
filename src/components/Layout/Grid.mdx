# GOV.WALES Layout

### Container

The default container is 960px wide with 20px left and right padding.\
A 100% container can be set by declaring that the container is fluid.

Rows will automatically add -20px margin left and right.\
Rows can be set to center or reverse content.

The maximum number of columns that can be applied to a row is 12.\
There is a gutter width of 20px on each side of a column.\
Columns can be displayed in reversed or ordered first or last.
