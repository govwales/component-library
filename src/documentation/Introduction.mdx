# GOV.WALES Component Library

> This component library is in development and is subject to change.

<br />

## About

This library contains GOV.WALES design system components built using [Typescript React](https://react.dev/learn/typescript) and [SASS](https://sass-lang.com/dart-sass/).

## Usage

1. Install with NPM

```
npm install @govwales/component-library
```

2. Importing GOV.WALES React components

```
import { ComponentName } from '@govwales/component-library';
```

For more information refer to the [Github](https://github.com/govwales/govwales-component-library) repository.
